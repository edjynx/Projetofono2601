# Fonoedv - Landing Page

Uma landing page profissional, responsiva e acolhedora para uma clínica de fonoaudiologia (Fonoedv). O objetivo deste projeto é apresentar os serviços prestados, as profissionais envolvidas e os depoimentos de pacientes, transmitindo confiança e facilitando o contato.

## 🚀 Tecnologias Utilizadas

Este projeto foi desenvolvido com as seguintes tecnologias:

*   **HTML5**
*   **CSS3** (Puro, sem frameworks)

## 💻 Como executar o projeto

Para visualizar o projeto localmente, basta clonar este repositório e abrir o arquivo `index.html` em seu navegador de preferência.

```bash
# Clone este repositório
git clone https://github.com/edjynx/fonoedv.git

# Acesse a pasta do projeto
cd fonoedv
```
Dê um duplo clique no arquivo `index.html` ou utilize alguma extensão como o Live Server no VS Code para iniciar um servidor local.

## 📱 Estrutura do Projeto

A landing page possui as seguintes seções principais:
*   **Cabeçalho e Banner Principal:** Introdução da clínica e chamada para ação (Call to Action).
*   **Serviços:** Detalhamento dos tratamentos e serviços oferecidos.
*   **Sobre:** Apresentação da profissional / clínica.
*   **Depoimentos:** Opiniões de pacientes para construir credibilidade.
*   **Rodapé:** Informações de contato e links úteis.
